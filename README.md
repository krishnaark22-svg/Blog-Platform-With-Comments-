📘 Blog Platform with Comments
A full-stack web application where users can create, manage, and interact with blog posts through comments. This project demonstrates core full-stack development concepts including authentication, REST APIs, and database integration.
🚀 Features
🔐 User Registration & Login
✍️ Create, Edit, Delete Blog Posts
💬 Comment on Posts
🔗 RESTful API Integration
🗄️ MongoDB Database
🛠️ Tech Stack
Frontend
HTML
CSS
JavaScript
Backend
Node.js
Express.js
Database
MongoDB
🔌 API Endpoints
Auth
POST /api/auth/register → Register user
POST /api/auth/login → Login user
Posts
GET /api/posts → Get all posts
POST /api/posts → Create post
DELETE /api/posts/:id → Delete post
Comments
POST /api/comments → Add comment
GET /api/comments/:postId → Get comments
